Data Immersion & Wrangling
This project focuses on acquiring, cleaning, and preparing raw data for analysis using Python and Pandas.

📌 Project Description

The system takes a CSV dataset (e.g., sales/customer transactions) as input, performs data profiling to identify missing values, duplicates, inconsistent formatting, and outliers, then applies cleaning and transformation techniques to generate a final analysis-ready dataset.


🚀 Features

Upload dataset (CSV file)
Automatic data profiling
Missing value detection & handling
Duplicate removal
Standardization of date formats
Data type correction
Feature engineering (e.g., Customer Age from Date of Birth)
Creation of cleaned dataset output
Data Dictionary documentation

🧠 Tech Stack

Python
Pandas
NumPy
Matplotlib
